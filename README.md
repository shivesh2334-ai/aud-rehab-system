# AUD Clinical Management System

A specialized clinical decision support tool and management system for **Alcohol Use Disorder (AUD)**. This application streamlines the assessment, withdrawal management, and long-term recovery tracking for patients in rehabilitation settings.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fyour-username%2Faud-rehab-system)

## üöÄ Key Features

-   **PAWSS Risk Assessment**: Integrated Prediction of Alcohol Withdrawal Severity Scale (PAWSS) calculator.
-   **Phase-Based Protocols**:
    -   **Assessment**: Tracking consumption habits and medical comorbidities (HbA1c, BP).
    -   **Withdrawal**: Dynamic protocols for high-risk (Inpatient) vs. low-risk (Outpatient) scenarios.
    -   **Treatment**: Clinical guidelines for first-line pharmacotherapy (Naltrexone, Acamprosate) and psychosocial interventions.
-   **Progress Tracking**: Clinical note-taking system and long-term follow-up scheduling.
-   **A4 Optimized Export**: Professional "Export to PDF" functionality designed for medical records.
